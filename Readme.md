# Requests
A simple command-line API get requests script
## Description
I built this while learning python to practice making API get requests

## Features
it features all the basic functions a calculator needs.
1. API Dashboard
2. Country lookup
3. Random Cat_fact generator


## Technologies Used
- Python 3
- Git

## How to Run
```bash 
python api_dashboard.py

python country_lookup.py

python cat_fact.py

```
